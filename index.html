Comment ça marche

1
Capture Hydratrade
Prends une capture d'écran de Hydratrade et dépose-la dans la zone, ou clique pour parcourir. La capture reste sauvegardée localement (même après rechargement).
2
Analyse 100% automatique
Le site récupère les bougies 15m/1h + DXY, calcule un score technique, applique les garde-fous (session, jour, volatilité) et affiche un verdict BUY ou SELL. Tu peux aussi cliquer sur le bouton Analyser maintenant en haut de page pour relancer manuellement. Pas de cooldown — tu peux relancer quand tu veux.
3
Analyse 100% automatique
L'analyse technique se lance automatiquement au chargement de ta capture (environ 400 ms après l'upload). Tu n'as rien à cliquer. Pour relancer avec une nouvelle capture, utilise le bouton Remplacer la capture. Si les proxies CORS sont down, configure une clé Twelve Data dans les Réglages pour un flux fiable.
4
Sync entre appareils
Dans Réglages, utilise Exporter pour télécharger un JSON de tes données (réglages + capture + historique), puis Importer ce fichier sur un autre navigateur ou ordinateur pour synchroniser.
Règle d'or : le site fait tout le technique automatiquement. Pour une seconde opinion visuelle (chandeliers, structure), glisse ta capture dans notre chat Mavis.
